# SupplySync360 LinkedIn MD Pack

This pack contains updated LinkedIn markdown files for posts 01–17.

## Structure used in every post file
- `## Intent`
- `## Visible Text`
- `## LinkedIn Post Text`
- `## Image Brief`

## Important usage rule
- `Visible Text` is the only text that should appear on the generated image.
- `LinkedIn Post Text` is the LinkedIn caption text and should not be placed on the image.
- Asset format / frame text has been removed from `Visible Text` so it does not appear on the image.
- Each post is now structured as a multi-page carousel using 4 or 5 separate 4:5 portrait images.
- `Image Brief` explicitly tells the image generator to create separate images, one per page, for you to combine afterwards.

## Supporting files
- `00-body-content-index.md` = all posting captions in one file.
- `00-manifest.csv` = file list with page counts.
