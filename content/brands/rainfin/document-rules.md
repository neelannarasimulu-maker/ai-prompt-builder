# RainFin Document Rules

Page setup:
- A4 portrait unless another output format is explicitly selected
- White or soft mist document canvas
- Generous print-safe margins
- Use a premium advisory-forum document style

Header:
- Repeat on every page
- Use the official RainFin logo where available
- Use RainFin navy #173A59 for header text
- Keep the header clean, compact and advisory-led

Footer:
- Repeat on every page
- Include document name, confidentiality/status text where supplied, and dynamic page numbers
- Use small grey-blue text #5B6870

Section formatting:
- Main headings: RainFin navy #173A59
- Subheadings: deep teal #063543 or rain blue #1EA7E1
- Use AVBOB green #007A33 and Ninety One gold #F2A51A only as restrained partner/status accents when relevant
- Avoid overly dark backgrounds, black-heavy styling and dashboard clutter

Document generation:
- Use Body Content as the exact source of truth
- Preserve section order, labels, tables, rows and notes
- Do not summarise, remove, merge or rewrite content
