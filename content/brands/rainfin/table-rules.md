# RainFin Table Rules

Table format:
- Render Markdown pipe tables as real Word/PDF tables
- Do not convert tables into paragraphs
- Preserve blank cells as empty completion fields

Header row:
- Background: #173A59
- Text colour: #FFFFFF
- Font weight: Bold
- Font size: 9.5 to 10 pt

First column / label cells:
- Background: #F5FAFC
- Text colour: #173A59
- Font weight: Semibold

Input / response cells:
- Background: #FFFFFF
- Text colour: #26343A

Borders:
- Thin pale blue-grey borders
- Avoid heavy gridlines

Spacing:
- Keep rows readable and page-safe
- Wrap long text naturally
- Repeat header rows when tables flow across pages where possible
