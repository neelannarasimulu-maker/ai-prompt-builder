# Thenga Document Rules

Use these rules only for Word/PDF-style document outputs.

## Page setup

- A4 portrait unless the content file specifies otherwise
- White or Soft Sand #F7F1E3 page canvas
- Generous margins
- Print-readable spacing
- Clear section hierarchy

## Header

Use the Thenga logo where available. For light document pages, use:
`content/brands/thenga/assets/thenga-logo-transparent-light.png`

Header text may read:
`THENGA SOCIAL ENTERPRISES | Trust Infrastructure for Participation`

## Footer

Use a restrained footer with the document title, confidentiality/status wording where supplied by the content file, and dynamic page numbers.

## Document style

- Use deep navy section headings
- Use trust teal subsection headings
- Use soft sand or white input areas
- Use restrained gold or earth green accents
- Keep the document clean, premium, credible and investor-ready

## Source rule

For document outputs, use `## Body Content` as the source of truth. Do not use visual-only sections as document body content.
