# Thenga Visual Rules

Use a premium, human and investor-ready visual system built around trust, participation and value creation.

## Visual style

- Use layered trust architecture, hub-and-spoke ecosystems, participation pathways and value-exchange flows.
- Use human participation imagery carefully: citizens, creators, farmers, healthcare workers, businesses, communities and partners.
- Use diagrams that feel operational and strategic, not decorative.
- Use soft but rich brand-aware backgrounds with depth, not black-heavy cyber styling.
- Keep content panels readable with white, soft sand or mist card surfaces.
- Use Prosperity Gold, Earth Green, Ubuntu Purple and Trust Teal as meaningful accents.

## Background guidance

Use soft brand depth: deep ubuntu navy or trust teal with warm sunrise, soft sand or earth-tone balance. Avoid backgrounds that are too dark, too flat, washed out or neon.

## Visual typography

Use bold executive sans-serif headings, compact card labels and readable body text. Do not overload a visual with paragraph blocks.

## Logo placement

Use the Thenga logo in a clean header area. Keep it subtle and secondary to the main message unless the output is a cover or opening visual.

## Avoid

- Generic fintech dashboards
- Crypto coin clichés
- Fake transaction screens
- Unsupported metrics
- Overly dark sci-fi control-room visuals
- Repeating detailed spoke descriptions unless the content file requires them
