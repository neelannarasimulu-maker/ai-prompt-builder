# Thenga Table Rules

Use these rules for Word/PDF document tables.

Table header row:
- Background: #102A43
- Text colour: #FFFFFF
- Font weight: Bold

First column / label cells:
- Background: #F7F1E3
- Text colour: #1E2A32
- Font weight: Bold

Input/data cells:
- Background: #FFFFFF
- Text colour: #1E2A32

Accent rows or status cells:
- Trust / Verified: #008B8B
- Prosperity / Value: #D6A11E
- Impact / Growth: #3A7D44
- Community / Inclusion: #55299B

Borders:
- Thin light grey borders
- Avoid heavy gridlines

Spacing:
- Keep rows tall enough for completion fields
- Avoid clipped text
- Keep tables print-readable
