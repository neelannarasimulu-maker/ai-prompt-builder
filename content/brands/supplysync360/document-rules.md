# SupplySync360 Document Rules

Page setup:
- A4 portrait unless another output format is explicitly selected
- White, mist or soft grey document canvas
- Generous print-safe margins
- Use a clean executive operational-technology document style

Header:
- Repeat on every page
- Use the official SupplySync360 logo where available
- Use deep teal #006C70 for header text
- Keep the header precise and understated

Footer:
- Repeat on every page
- Include document name, brand footer where supplied, and dynamic page numbers
- Use small grey-teal text #5D6D6D

Section formatting:
- Main headings: deep teal #006C70
- Subheadings: vivid teal #008B8B
- Use signal yellow #FFD700 only as a restrained action or highlight accent
- Avoid fake dashboards, clutter and generic icons in document outputs

Document generation:
- Use Body Content as the exact source of truth
- Preserve section numbering, labels, tables and lists
- Do not summarise, remove, merge or rewrite content
