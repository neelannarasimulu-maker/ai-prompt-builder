# BMA/Open Table Rules

Table format:
- Render Markdown pipe tables as real Word/PDF tables
- Do not convert tables into paragraphs
- Do not use HTML table tags
- Preserve blank cells as empty completion fields

Header row:
- Background: #0B2F44
- Text colour: #FFFFFF
- Font weight: Bold
- Font size: 9.5 to 10 pt

First column / label cells:
- Background: #EAF5F3
- Text colour: #243238
- Font weight: Semibold

Input / response cells:
- Background: #FFFFFF
- Text colour: #243238
- Font weight: Regular

Borders:
- Thin light grey borders
- Avoid heavy gridlines
- Keep tables crisp and printable

Spacing:
- Use enough row height for form completion fields
- Do not clip text
- Allow long guidance text to wrap naturally
- Split very long tables across pages cleanly with header rows repeated where possible
