# BMA/Open Document Rules

Page setup:
- A4 portrait unless another output format is explicitly selected
- White or very soft mint document canvas
- Margins: generous and print-safe
- Use a clean executive document layout
- Prioritise readability over decorative effects

Header:
- Repeat on every page
- Use the official BMA/Open logo where available
- Header text: BLOCK MARKETS AFRICA / OPEN | New Client Business Case Template
- Use navy #0B2F44 for header text
- Keep header compact and professional

Footer:
- Repeat on every page
- Footer text: Block Markets Africa / Open | Confidential business development template | For deal qualification only
- Include dynamic page numbers on every page
- Use small grey text #566368

Section formatting:
- Use navy #0B2F44 for main section headings
- Use deep teal #006C70 for subsection headings
- Use green accents #67B345 or #56C947 only for action emphasis or status markers
- Avoid hype, decorative clutter and generic fintech dashboard styling

Document generation:
- Use Body Content as the exact source of truth
- Preserve section numbering and heading order
- Preserve all wording, labels, options, rows, scoring values and checklist boxes
- Preserve empty cells as blank completion fields
- Do not summarise, remove, merge or rewrite content
